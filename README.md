# OuiCestFait API MVP

## Lancement rapide
1. Double-clic sur run.ps1 (ou clic droit > Exécuter avec PowerShell)
2. Ouvre http://127.0.0.1:8000/docs

## Endpoints clés
- POST /mission/demande : créer une mission
- POST /mission/validation : valider une mission
- GET /health : vérifier que tout tourne
